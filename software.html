▼Project 1: Secured Authentication Process for Physically Challenged People Using Morse Code and Machine Learning Algorithm

Objective: Developed a secure and accessible authentication system for individuals with motor disabilities using eye blinks to input Morse code as passwords.
Technologies Used: Implemented Haar Cascade algorithm for facial and eye detection, and Dlib library for tracking facial points and eye aspect ratio (EAR).
Features: Created a virtual keyboard for Morse code input controlled by eye blinks, enabling contactless password entry.
Challenges Addressed: Tackled issues such as time consumption of Morse code entry, visibility under low light conditions, and usability for individuals with partial blindness.
Achievements: Achieved high accuracy in real-time eye tracking and Morse code conversion, enhancing security and usability for physically challenged users.
