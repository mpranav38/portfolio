▼Project 3: Understanding the Cost of Cloud Computing

Objective:This project aims to assess the costs and profitability of cloud computing versus private cloud infrastructure for a startup company across three configurations: Hadoop/Spark cluster, MacOS application development, and Ethereum cryptocurrency. Features: Implemented functionalities to download eBooks, read them, manage favorites, toggle dark mode, and swipe to delete downloads.
Technologies Used: The project utilizes AWS calculators, ThinkMate for server pricing, Apple Website for Mac Mini pricing, and cryptocurrency mining calculators to estimate costs and profitability.
Challenges Addressed: The key challenges include accurately estimating cloud and hardware costs, analyzing cryptocurrency mining profitability, and determining whether renting or buying infrastructure is more cost-effective.
Outcome The project will produce a detailed report comparing the 5-year costs and profitability of public cloud services (AWS) versus private cloud infrastructure, providing a recommendation on the best investment strategy.

Challenges Faced:
Ensuring efficient parallel processing and synchronization across multiple threads.
Managing memory usage effectively to handle large datasets.
Optimizing the performance of hash generation, sorting, and disk writing operations.
